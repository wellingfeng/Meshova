# Cropout 岛屿生成复刻笔记

参考合集：一只小狍子啊《虚幻5 Cropout 示例项目教程》，本地归档位于 `ref/bilibili-cropout-series/`。

核心三集：

- `BV1h94y1p7zf`：建立动态网格组件，用多个扁平圆片累积岛屿体块。
- `BV1cN411n7Cf`：融合圆片边界、平滑网格，按高度/法线区分顶面与侧面，叠出草地和沙岸。
- `BV15QCNYdE8P`：完成岩层、沙滩、草地三级轮廓，按顶面区域散布植被和岩石实例。

Meshova 对应实现：

1. 确定性生成一组互相重叠的圆形叶瓣。
2. 沿极角求圆并集最远交点，得到连续、可焊接的岛屿轮廓。
3. 对同一轮廓做不同偏移和高度，生成岩层、沙滩、草地三层动态网格。
4. 在草地安全内区采样，生成树木和岩石。
5. 同一生成核通过种子、岛数、叶瓣数、拉伸、海岸宽度产生六个模型库预设。

实现不复制 Unreal 蓝图或 Cropout 资产；仅复现视频展示的程序化方法与视觉结构。
