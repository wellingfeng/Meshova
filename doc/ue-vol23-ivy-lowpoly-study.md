# UE5 VOL23 Ivy Low Poly 学习与程序化复刻

## 资产证据

- 参考包含 49 个常春藤静态网格：`01a–01o` 15 个，`02a–02ah` 34 个。
- 每个网格有 4 级 LOD。LOD0 约 30–1355 三角面，主体尺寸约 0.5–4 米；5 个稀疏藤条可延伸至约 12 米宽、7 米高。
- `01` 族混合活叶材质与独立木质藤条；`02` 族统一使用 Ivy Atlas，覆盖墙面团簇、悬垂、横爬、帘状、稀疏五类剪影。
- Atlas 同时提供活叶、枯叶、枝条卡片、单叶变体；材质含 Albedo、Normal、Height、Opacity、RHA/RMA、Subsurface，并使用双面叶片、风摆、透明裁切。
- 叶形不是椭圆：明显心形叶基、尖顶、浅三裂，叶脉高对比。枝条细、分叉少，轮廓主要由叶片密度与方向控制。

## Meshova 复刻映射

| 参考规律 | 程序化实现 |
| --- | --- |
| 49 个固定静态网格 | 种子驱动的无限变体 |
| Atlas 枝叶卡片 | 实际低模藤茎与三裂叶几何 |
| 墙面、悬垂、横爬、帘状、稀疏 | `wall`、`hanging`、`runner`、`curtain`、`sparse` |
| 活叶与枯叶 Atlas | 成熟、嫩绿、枯黄三组材质 |
| 4 级手工 LOD | `lod=0..3` 同时降低茎截面、路径段数、叶密度 |
| UE 风摆材质 | 叶片 `windWeight=1`，藤茎保留渐变权重 |

实现位于 `src/models/ivy-lowpoly-kit.ts`。输出保持确定性，不复制原网格或贴图数据。
