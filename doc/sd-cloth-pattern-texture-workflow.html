核心结论

服装花纹不要当成一张位图。应拆成 织物底纹、装饰图案 mask、 变形/重复规则、height/normal/roughness 联动 四层。 AI 生成流程也应优化这些参数，而不是直接优化像素。

程序化优先 可平铺 mask 驱动 高度分层 粗糙度相关 VLM 语义校验

从教程抽象出的模式

SD 节点名不是重点。重点是图案如何被拆层、重复、扰动、再映射到 PBR 通道。

织物底纹 Base weave

横向/纵向细线 + 拉伸噪声 + 轻微 warp。底纹负责“布”的频率，不负责大花纹。

花纹 mask Motif mask

花、藤蔓、几何纹先做黑白 mask，再决定它是印花、提花、刺绣还是金线。mask 复用到颜色、高度、粗糙度。

重复与错位 Tile Sampler

图案靠格子重复、偏移、旋转、缩放变化产生“布料纹样”。随机必须可 seed，边界必须可平铺。

方向变形 Directional Warp

花纹不能像贴纸。用低频布纹/褶皱场去扰动 mask，让图案贴住织物流向。

高度混合 Height Blend

布纹微高度、花纹中高度、刺绣/金线高高度分开叠。法线从 height 来，避免 baseColor 硬转 normal。

通道联动 PBR coherence

同一 mask 同时影响 baseColor、roughness、height、AO。否则看起来像彩色图片贴在塑料上。

推荐生成管线

适合 Meshova：脚本优先、seed 确定、headless 截图迭代。

1. 语义拆解

VLM/文本：布料类型、纹样家族、重复方式、主辅色、工艺。

2. 参数草案

输出参数，不输出位图：密度、方向、warp、motif、height、roughness。

3. 程序生成

调用 Meshova texture 函数生成 baseColor/roughness/normal/height。

4. 材质预览

球体 + 布料 drape 双预览。只看平面会漏掉法线和高光问题。

5. 自动评分

无缝、频谱、语义、PBR 合法性、通道相关性。

6. 黑盒优化

优化参数，不优化像素。保留可编辑程序图。

参数 schema 建议

这些参数应暴露给 AI 迭代器。范围可限制，避免生成发散。

参数组字段用途建议范围
fabricweaveType, threadDensityU/V, threadAngle控制平纹、斜纹、缎纹、针织等底层结构。密度 16-160；角度 0-180°。
fiberfiberScale, fiberStretch, fiberNoise产生布料微纤维，影响 height/roughness。stretch U/V 分开；noise 强度 0-0.25。
motifmotifFamily, repeatU/V, symmetry, motifScale花、藤、几何、点阵、条纹、格纹等装饰 mask。repeat 1-24；scale 0.15-1.5。
distortionwarpScale, warpStrength, warpDirection让图案跟随织物流向，避免贴纸感。strength 0-12 px；低频优先。
layeringprintDepth, embroideryDepth, edgeBevel决定印花/提花/刺绣差异。印花接近 0；刺绣最高。
colorbasePalette, motifPalette, colorBleed颜色只由 mask 和调色板映射，避免直接烘参考图。2-5 色；bleed 0-0.2。
pbrroughnessBase, roughnessDelta, normalStrength, metallicMask通道联动。布料 metallic 默认 0，金线/烫金局部可开。roughness 0.55-0.95；normal 0.5-4。

Meshova 落地建议

现有 texture 模块已经覆盖很多 SD 等价能力。补缺点集中在服装纹样 preset 和更强的 motif 布局。

P0：新增 cloth material builder

做 clothWeave()、jacquard()、printedFabric()、embroideredFabric()。输出 MaterialFields，接 materialFromFields。

P0：通道相关性约束

同一个 motif mask 驱动 baseColor/roughness/height。加测试：花纹区 roughness/height 差异可解释，metallic 只在金线工艺出现。

P1：增强 Tile Sampler

当前已有 tileSampler。建议加 stagger、mirror、brick offset、row/column seed、per-instance mask 输出，支持织锦/壁纸式连续花纹。

P1：各向异性噪声

服装纹理常是拉长纤维。新增 anisotropicFbm 或给现有 fBm 包一层旋转/缩放 UV。

P2：VLM 参数提取

参考图只做语义拆解：布料类型、花纹类型、重复密度、颜色、工艺。禁止直接把参考图转贴图。

P2：评分闭环

截图送 VLM + 本地指标。优化参数，回写 seed 和 schema。失败样例保存到 out/。

工艺类型与通道策略

同样是“花纹”，PBR 策略不同。

印花布 Printed

花纹主要进 baseColor。height 很低，roughness 小幅变化。边缘可有轻微渗色/褪色。

提花 / 织锦 Jacquard

花纹和底布共享织物方向。motif 改变纱线亮度、粗糙度、微高度，不像油墨盖在上面。

刺绣 Embroidery

motif 高度最高。花纹内部要有独立 thread direction，边缘 bevel 明显，AO 加强。

烫金 / 金线 Metallic accent

base fabric metallic=0。只有金线 mask 局部 metallic=1 或高 F0，roughness 更低，normal 仍保留纤维。

格纹 / 条纹 Plaid

用横竖 stripe field 组合，不用图片。交叉处可颜色叠加，height/roughness 跟随纱线密度。

旧布 / 磨损 Worn cloth

磨损 mask 应受褶皱/边缘/高点影响。污渍、褪色走低频，不破坏基础可平铺。

自动评估指标

这些指标适合接 Meshova headless 截图闭环。

指标检查什么失败信号修正参数
Seam score左右/上下边界像素差、法线差、粗糙度差。平铺缝明显。提高 makeTile band，降低大尺度不连续 warp。
Frequency band低频花纹、中频编织、高频纤维是否分层。全图同频噪声，像脏墙。分离 motifScale、threadDensity、fiberScale。
Channel coherenceheight/normal/roughness 是否跟 motif mask 相关。颜色有花，法线无花；或 roughness 随机。用同源 mask 驱动多通道。
Material legality布料 metallic、roughness、normalStrength 是否物理合理。布像金属/塑料。metallic 归零，roughness 上调，法线强度限幅。
VLM descriptor预览图是否被描述成目标布料和目标花纹。目标“刺绣”，结果被看成“印花”。提高 embroideryDepth、edgeBevel、threadDirection。
Readability远看仍能认出大花纹，近看有纤维。花纹碎、糊、太密。降低 repeat，提高 motif contrast，减少 warpStrength。

反模式

这些会破坏 Meshova “程序化 + 可迭代”的目标。

直接图生贴图

像素看似接近，但不可编辑、不可泛化、不能稳定平铺。只允许参考图做语义和参数提取。

从 baseColor 猜 normal

颜色边界不是高度边界。normal 应由 height stack 生成，颜色只是外观层。

roughness 独立噪声

粗糙度和纤维/花纹无关时，高光会乱。roughness 应绑定 weave/motif/wear mask。

把褶皱烘进平铺材质

服装大褶皱属于几何或局部 mask。平铺材质只放微褶和布纹，避免重复假褶。

无限制 warp

warp 过强会毁花纹识别。大花纹轻扰动，高频纤维可强扰动。

全图单一尺度

真实布料有三层尺度：图案、编织、纤维。缺一层就假。

下一步实施清单

按收益排序。

顺序任务验收
1加 printedFabric 和 jacquardFabric 两个 preset。生成 baseColor/roughness/normal/height，validateMaterial 通过。
2补 anisotropicFbm，或封装旋转/缩放 UV 的噪声采样。能生成横向、纵向、斜向纤维；seed 稳定。
3增强 tileSampler：stagger/mirror/brick offset。能做格纹、提花重复、错列花纹。
4加 cloth preview：球体 + 布料垂坠 mesh 双视图。headless 截图能同时看平铺和衣服表面效果。
5加评分：seam、frequency、channel coherence、VLM descriptor。AI 能自动调参，不靠人工肉眼每次看。
