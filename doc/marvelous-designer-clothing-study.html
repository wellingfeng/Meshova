核心结论

Marvelous Designer 最值得学的不是 UI，也不是专有模板，而是“衣服 = 2D 纸样 + 缝线图 + 人体量体 + 3D 初始摆放 + 布料物理 + 分层约束”。Meshova 若要生成常见衣服，最好先建 pattern-space DSL，再把纸样转成可渲染 mesh；不要一开始直接手写衣服网格。

MD 概念对 Meshova 的抽象原因
Pattern Drafter参数化服装模板生成器衣服可由量体点和少量风格参数生成。
3D Pen / Avatar Line在 avatar 表面画 styleline，再展成纸样图片/草图输入可以先变成 avatar 上的曲线。
Sewing缝线图：边到边、多边到一边、方向、缩褶衣服拓扑来自缝线，不来自三角面随机拼接。
Arrangement Points初始摆放场：身体部位锚点、包围体、朝向仿真稳定性强依赖初始位置。
Particle Distance衣服 LOD / solver 分辨率快速迭代与高质量输出分离。
Fit Maps质量评分：拉伸、压力、穿模、松量给 AI 闭环可量化反馈。

关键学习点

纸样优先
MD 新版 Pattern Drafter 允许从服装类型和量体生成可编辑纸样；AI Pattern Drafter 已支持 T-Shirts、Pants、Skirts。Meshova 可从同三类开始。
线驱动剪裁
3D Pen / Avatar Line 能在 avatar 上画线并提取 pattern。Meshova 可把“衣领线、袖窿线、下摆线”作为第一等输入。
缝线是拓扑
Segment / Free / 1:N / M:N sewing 都暗示一个 seam graph。生成衣服时先维护 graph，再生成三角网格。
粗细分阶段
MD 建议制作/试穿阶段用较大 Particle Distance，完成后降到更小值提质。Meshova solver 应支持低分辨率搜索 + 高分辨率烘焙。
分层是硬约束
内搭、外套、口袋、纽扣等需要 layer order。没有层，仿真和投影都会错。
生产资产要 retopo
MD 有 quad remesh / retopology / UV 编辑。Meshova 出游戏资产也要把仿真网格转成稳定拓扑和可控 UV。

建议数据模型

Meshova 可新增 src/clothing/ 或 src/garment/。底层不依赖 Three.js，保持纯数据：2D 纸样、缝线、量体、布料参数、生成后的 mesh。
type GarmentTemplate = {
  id: string
  family: "top" | "bottom" | "dress" | "outerwear" | "accessory"
  measures: MeasureBinding[]
  params: ParamDef[]
  panels: PanelDef[]
  seams: SeamDef[]
  arrangement: ArrangementDef[]
  layers: LayerDef[]
  fabric: FabricPhysical
}

type PanelDef = {
  id: string
  mirror?: "left" | "right" | "center"
  grain: Vec2
  edges: PatternEdge[]       // line / quadratic / cubic
  darts?: DartDef[]
  pleats?: PleatDef[]
  internalLines?: InternalLineDef[]
}

type SeamDef = {
  a: EdgeRef
  b: EdgeRef
  direction: "same" | "opposite"
  ratio?: number             // gathers / easing / 1:N sewing
  seamAllowance?: number
  puckering?: number
}
结构字段实现要点
Panel闭合 2D 曲线、边序、grain、语义标签边必须有稳定 ID。后续参数、缝线、UV 都引用边。
PatternEdgeline / Bezier、长度、曲率、局部方向支持按边长度、曲率、端点法向改变，贴合 GarmentCode 类模板思想。
SeamEdgeRef 对、方向、比例、松量、缝份允许一对一、一对多、多对多。先支持一对一和一对多。
MeasureBinding胸围、腰围、臀围、肩宽、袖长等参数默认从 avatar 自动量取，可被用户覆盖。
Arrangement身体部位锚点、包围体、初始旋转、offset无 cloth solver 时也可作为 surface wrap 初始位置。
Layer内外层序、碰撞厚度、临时仿真层口袋/外套/内搭必须有。
FabricPhysicalstretch、bend、shear、buckling、density、damping、pressure先用于视觉 preset，后用于 XPBD 约束。

参数规则

长度类
衣长、袖长、裤长、裙长、领高、帽深。实现：沿边方向或指定 2D 方向伸缩。
围度类
胸围、腰围、臀围、腿围、袖口、裤脚。实现：左右对称扩边，保持 seam 对边约束。
曲率类
袖窿、领口、腰线、公主线、下摆弧。实现：Bezier 控制点偏移。
结构类
开衩、门襟、拼片数、口袋、拉链、纽扣。实现：可选 panel + trim graph。
褶皱类
百褶、抽褶、松紧腰、袖口收口。实现：seam ratio + fold lines + wrinkle field。
版型类
修身、宽松、oversize、A 字、直筒、阔腿。实现：ease preset 影响多组围度和 flare。

物理仿真与工程近似

Meshova 目标是程序化资产生成，不必第一版复刻 MD solver。建议三阶段：先形状近似，再 XPBD，再 WebGPU 加速。
阶段做法能生成限制
Stage A：无仿真 用 avatar 胶囊/截面曲线直接生成衣服壳体，panel 只作为 UV 和 seam 语义。 T 恤、背心、裙子、裤子、披风轮廓。 褶皱假，垂坠弱，但快、稳定、适合 AI 优化。
Stage B：启发式 drape panel triangulate -> 投影到 avatar 外侧 -> seam shrink -> 重力下垂 -> wrinkle displacement。 宽松袖子、裙摆、裤脚、帽兜、外套下摆。 不保证物理正确，但视觉能过 demo。
Stage C：XPBD cloth stretch / shear / bend / seam / collision / layer / pressure 约束，低分辨率搜索，高分辨率烘焙。 自然垂坠、衣摆碰撞、多层衣服、软布料差异。 实现成本高，需要稳定碰撞和收敛策略。

XPBD 约束清单

Stretch
经向/纬向边长约束，对应 warp/weft 伸缩。
Shear
三角面斜向剪切约束，控制布料斜向变形。
Bend
相邻三角二面角约束，控制硬挺/柔软。
Seam
两条 pattern 边采样点互相拉近，可带 ratio 形成抽褶。
Collision
avatar 胶囊/SDF 碰撞，外层衣服对内层碰撞。
Pressure
羽绒、枕状、充气、厚衣服体积感。
Pin / Tack
固定点、临时缝合、挂点、肩带。
Layer
内外关系，防止口袋、纽扣、外套进入内层。

质量评分

穿模率
衣服顶点在 avatar SDF 内部的比例和最大深度。
拉伸率
边长相对 rest length 的百分比，类似 strain map。
压力/紧身
碰撞接触密度 + 法向推力近似，帮助判断“穿不了”。
对称误差
左右 panel 或渲染剪影的镜像差，适合 T 恤/裤子。
缝线闭合
seam pair 采样点距离均值和最大值。
视觉目标
多视角 silhouette IoU + VLM 分类一致性 + 关键部位检测。

布料与 PBR 材质

MD 的物理属性和 Meshova 现有程序化 PBR 可以分两层：FabricPhysical 控制形变，FabricMaterial 控制视觉。两者用同一 preset 名称绑定。
MD 学到的参数Meshova 字段视觉/物理影响
Stretch Weft/WarpstretchWeft, stretchWarp针织、弹力布更容易贴身；非弹力棉布更容易形成紧绷。
ShearshearStiffness剪切高更容易出硬折，低更软更垂。
Bending Weft/WarpbendWeft, bendWarp牛仔、皮革硬；丝绸、雪纺软。
Buckling Ratio/StiffnessbuckleRatio, buckleStiffness折角形成阈值和折痕硬度。
Density/Weightdensity重布下垂更明显。
Pressurepressure羽绒服、坐垫、充气衣物。
ShrinkageshrinkWeft, shrinkWarp水洗缩水、收口、内层贴身。
LayerlayerIndex外套、口袋、装饰件的正确叠放。

Meshova 落地路线

里程碑要做验收
M1：Pattern Core 2D panel 曲线、边 ID、Bezier 采样、面积/自交检查、边长约束。 能生成 T 恤/裙子/裤子纸样 SVG 或 viewer overlay。
M2：Seam Graph edge ref、方向、1:N sewing、gather ratio、seam allowance。 可显示缝线配对，错误 seam 能报诊断。
M3：Avatar Measures 胶囊人体、胸/腰/臀/肩/臂/腿量取，template 参数绑定。 换身材后衣服围度随动。
M4：Heuristic Drape pattern triangulate、投影壳体、重力下摆、褶皱 displacement、碰撞修正。 首批 3 类衣服在 viewer 内可调参数并截图。
M5：Fabric Presets cotton/denim/wool/leather/silk/nylon physical + PBR 绑定。 同一 T 恤换材质能明显改变视觉和形变倾向。
M6：AI Closed Loop VLM 分类 -> 模板选择 -> 参数优化 -> silhouette/VLM 分数。 输入常见服装图，生成同类衣服，袖长/领型/裤型基本对。
M7：XPBD Solver stretch/shear/bend/seam/collision/layer/pressure，低高分辨率两阶段。 宽松裙摆、袖子、外套下摆有可信垂坠。
M8：Game Mesh Export quad-ish remesh、UV from panels、trim geometry、LOD、OBJ/GLTF。 导出可进引擎，材质通道齐，拓扑稳定。

优先做

  • pattern.ts：2D 曲线、panel、edge loop、self-intersection。
  • seam.ts：缝线图、配对验证、采样点。
  • templates/tshirt.ts、skirt.ts、pants.ts。
  • avatar/measures.ts：胶囊人体量体。
  • drape/heuristic.ts：先出能看的 3D mesh。

先别做

  • 不要照搬 MD 模板、资产、UI、商业库。
  • 不要从 Blender/MD GPL/专有代码抄实现。
  • 不要第一版追真实制衣生产尺寸。
  • 不要一开始做完整复杂外套、正装西装。
  • 不要只做 shader 褶皱，缺纸样会导致 AI 不可控。

细节库：结构件与衣服语义

领口 / Collar / Neckline
圆领前后领口 Bezier，rib collar ring，适合 T 恤。
V 领前中点下移，领口两侧直线或轻曲线。
高领neck band panel 高度增加，bend 稍高。
翻领collar stand + collar flap，需 fold line。
帽兜左右 hood panel + center seam + face opening，Tier 2。
袖子 / Sleeve
无袖删除 sleeve panel，袖窿边加 binding。
短袖sleeve cap + sleeve length，袖口围度。
长袖增加袖长，腕口 cuff，可设宽松。
泡泡袖sleeve cap gather ratio > 1，袖口收紧。
插肩袖肩线改为从领口到腋下的斜线，panel 拓扑不同。
裤装 / Pants
裆部front/back crotch curve 是裤子核心，需保持 seam 长度兼容。
裤型straight/slim/wide/jogger 主要改 thigh/knee/hem 宽度曲线。
腰头waistband panel 环绕腰部，支持松紧或腰带。
口袋先做贴袋，再做插袋；贴袋可作为外层 panel。
裙装 / Skirt
直筒裙front/back 简单 panel，hipEase 小，长度可调。
A 字裙hemWidth > hipWidth，flare 参数明显。
百褶裙pleatCount、pleatDepth、fold direction，初期可 shader/geometry 假褶。
包臀裙fit/pressure 要严，容易穿模，需 slit 参数。
装饰件 / Trims
纽扣curve path 上 instancing 小圆柱，buttonhole 对应 internal line。
拉链zipper tape + teeth instances + slider mesh，沿 seam 或 internal line。
明线topstitch curve，沿 edge offset 生成细绳/贴图线。
抽绳curve rope + eyelets，hoodie/裤腰常用。
口袋patch pocket 是最小实现，welt pocket 后做。
