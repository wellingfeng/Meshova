# Bilibili 风格化岩石复刻笔记

- 来源：BV18A4m1579D《Houdini程序化生成风格化岩石》，jesselee阿布。
- 本地参考：`out/reference/BV18A4m1579D/reference.mp4`、`contact-sheet.jpg`、`key-sheet.jpg`。
- 造型主干：近方形浮岛轮廓；顶部平整；四周由竖向岩柱拼成；底部向中心收尖。
- 中层细节：宽窄不一的崖壁模块；局部前凸岩台；深色缝隙切断连续侧面。
- 顶部层次：主草盖加一至两级偏心岩台，形成旋转上升的台地节奏。
- 材质关系：冷灰岩体、深蓝灰凹缝、黄绿色草盖；色块大，微表面细节弱。
- Meshova 映射：确定性分块柱体生成崖壁；锥化柱体生成悬浮底部；噪声网格生成草盖；参数控制尺寸、高度、分块、阶数、破碎度、草边距。
