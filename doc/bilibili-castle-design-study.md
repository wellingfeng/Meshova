## 设计原则

1. 城堡是防御系统，不是单体高塔。外墙、内墙、院落、内堡形成逐层退守的纵深。
2. 门是最弱点。入口需要双塔门楼、落栅、狭长通道、桥梁或折线路径，让进攻方持续暴露。
3. 塔楼应凸出墙线。圆塔覆盖墙脚和相邻墙段，减少死角；角塔承担交叉火力。
4. 城墙顶部是交通系统。连续墙道、垛口和塔楼平台让守军快速调动，不只是轮廓装饰。
5. 内堡是终局据点。它应更高、更厚、入口更受控，并能俯瞰内外院落。
6. 院落有功能分区。兵营、厅堂、粮仓、水井和生活空间决定城堡能否承受长期围城。
7. 布局服从地形。山脊堡用高差和狭窄接近面；河关堡控制桥梁和水路；平地堡依赖同心环墙与护城河。

## 程序化映射

| 原理 | 生成规则 |
| --- | --- |
| 防御纵深 | 多边形幕墙环 + 内院 + 中央内堡 |
| 侧射火力 | 墙角圆塔凸出墙线，箭孔朝四向 |
| 强化入口 | 墙段留口，实例化双塔门楼、落栅、吊桥 |
| 连续调动 | 每段幕墙生成墙顶巡逻道和双侧垛口 |
| 围城生存 | 院内生成水井、粮仓、厅堂、兵营 |
| 地形适配 | 同心、山脊、河关三套独立平面拓扑 |

## 三套复刻

- 同心王堡：护城河、十边外墙、四边内墙、双重门楼、中央内堡。
- 山脊要塞：不规则长墙、单一狭窄入口、折线引道、制高点尖顶内堡。
- 河关城堡：河心岛、南北双门、双桥控制、拉长墙体和两岸通路。
