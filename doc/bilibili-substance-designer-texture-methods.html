推荐生成流程

Meshova 贴图生成应从“材质类型 preset + 结构图层”开始，不从一张 baseColor 反推全部通道。

1. 结构分类

识别材质族：岩石、织物、木、砖、道路、雪冰、头发、叶片、特效。决定主生成器。

2. 大形 height

Tile / Shape / Voronoi / Wave / FlowField 生成主轮廓。先管读形，不急着上色。

3. 中频破损

Directional Warp、Slope Blur、Distance/Bevel、裂缝 mask，打散重复和边缘。

4. 微细节

孔洞、颗粒、纤维、划痕、毛刺、污渍。微细节强度独立参数化。

5. 通道联动

height 派生 normal/AO/curvature；mask 同时驱动 baseColor、roughness、metalness。

6. 自动评估

检查 tiling seam、频谱重复、法线强度、粗糙度范围、VLM 材质语义。

贴图制作方法

每张卡：从 B 站 SD 教程抽象出的通用 recipe，可直接转成 Meshova preset 或 builder。

岩石 / 崖壁 / 石块 height first

做法：用多尺度 ridged noise / Voronoi 切出岩块，Slope Blur 做侵蚀，Directional Warp 拉断层方向，curvature 取边缘高光与尘土。

ridged fbm → levels → slopeBlur → warp → cracks mask → heightToNormal → AO/curvature → color ramp
Meshova`ridgedMultiFractal` + `voronoi(f2-f1)` + `slopeBlur` + `warp`；缺口：更强的分层 strata / erosion preset。
岩石思路岩石系列简易岩壁

砖墙 / 石墙 / 鹅卵石 tile + mortar

先做砖块 id mask，再用随机高度、倒角、边缘破损、砂浆层分离。风格化版本强化大形和色块，写实版本强化小裂纹和污渍。

brick/tileSampler → floodFillRandom → bevel edges → mortar mask → damage noise → color variation
Meshova`brick`、`tileSampler`、`floodFillRandom`、`bevel` 已有；应补“砖块 id → per-cell params”高级接口。
风格化砖墙石材砖墙破旧红砖破损墙面

木板 / 木纹 / 书架 wave grain

木材不是纯噪声。用 wave / ring 生成年轮方向，用 plank mask 分块，再叠裂缝、划痕、孔洞、边缘磨损。烧焦木板额外用黑化 mask 和炭裂纹。

wave/rings → warp by fbm → plank tile mask → scratches → holes → roughness from grain + wear
Meshova`wave`、`wood`、`lacqueredWood` 已有；应补 plank generator、年轮 flow distortion、burnt edge mask。
木材纹理木板材质风格化木板燃烧木板

织物 / 麻布 / 羊毛 weave field

主结构是经纬线交错：两组方向性条纹，交叉处 height 互相压盖，纤维噪声沿线方向拉伸。粗糙度高，法线细节小但密。

stripes X/Y → over-under mask → anisotropic fiber noise → small height → normal detail → high roughness
Meshova`stripes`、`fabric`、`velvet`、`plushFur` 可用；缺 weave-specific interlace、thread normal combiner。
布料大师课麻布料织物布羊毛拼接

蕾丝 / 刺绣 / 服装花纹 mask mapper

用 shape/tile sampler 生成花型 mask，再用 distance / bevel / directional warp 产生线边，刺绣方向来自花瓣局部角度或 flow map。

motif mask → tile sampler → distance/bevel → thread direction map → embroidery height → fabric merge
Meshova可先做 `patternMask + weaveBase + stitchEmboss` preset。关键是暴露图案密度、线宽、方向、底布强度。
蕾丝教程官方蕾丝刺绣蕾丝花瓣刺绣方向

头发 / 毛发贴图 flow strands

核心是 strand flow，不是随机线。粗束发片先定走向，中束控制发缕，小束给毛丝；颜色根部到发梢渐变，alpha/height/normal 同源。

flow field → strand bands → fine fibers → root-tip gradient → alpha/height split → normal combine
Meshova已有拉伸噪声和 fur surface；建议补 `flowStrands(seed, directionMap)`，给发片、毛绒、草叶共用。
毛发贴图毛发系列头发制作

道路 / 沥青 / 路线标记 aggregate

沥青由骨料颗粒和沥青胶结层组成。先散点石子，再用细颗粒填充；路线标记作为上层 mask，边缘磨损由噪声和车辙控制。

stone scatter → bitumen base → dust/grime → lane marking mask → edge wear → tire streak roughness
Meshova`tileSampler` + `voronoi` 可做骨料；应补 asphalt preset、paint marking layer、tire-wear mask。
道路全流程无限 tiling

水泥墙 / 混凝土 / 破损墙 stain layers

大面积低频脏污决定观感，小孔洞和裂缝决定真实感。破损墙要分 plaster、base concrete、exposed damage 三层，每层有独立 roughness。

low freq stains → pore noise → crack mask → chipped mask → exposed layer blend → AO darkening
Meshova`concrete` 已有基础；建议补 damagedWall builder：面层/底层/裂缝/剥落四图层。
写实水泥墙破损墙面高级材质实战

雪 / 冰 / 融雪 layered translucency

雪用 soft height + high roughness，融雪用湿润 mask 降 roughness，冰用透明裂纹层和内部气泡。法线不能太硬，否则像石头。

soft noise → melt mask → wet roughness drop → ice cracks → bubbles → shallow normal
Meshova已有 `ice`、`water`、`frostedGlass`；建议补 snowMelt preset 和 wetness 通道联动。
雪花雪景幻想冰层

叶子 / 摄影测量清理 photo assisted

图片不是直接当贴图终点，而是抽 mask：叶脉、边缘、斑点、透明度。程序化层负责变体、tiling、PBR 通道一致性。

photo mask extract → vein distance → color cleanup → translucency/roughness masks → procedural variation
Meshova未来图片输入流程可借鉴：VLM/分割提语义 mask，程序化 preset 生成可控 PBR。
真实叶子材质

特效纹理 / 燃烧 / 动态遮罩 packed channels

特效贴图重点是 RGBA 打包和可动画 mask。噪声流动、梯度、溶解边缘、火焰裂纹比静态 PBR 更重要。

noise fields → gradient masks → dissolve edge → channel pack RGBA → flipbook/scroll params
Meshova可加 effectTexture builder：输出多通道 mask，不一定走完整 PBR。
动态特效贴图特效纹理燃烧材质

风格化材质 / 手绘感 shape readability

减少高频真实噪声，强化大色块、明确边缘、可读轮廓。色彩用 gradient/ramp 分层，不追求照片级粗糙度细节。

simple shape mask → bevel → handpaint ramp → controlled edge wear → limited micro noise
Meshova加 `style` 参数：realistic/stylized。控制噪声频率、色阶数、边缘宽度、AO 强度。
风格化砖墙手绘/Ghibli风格化展示

金属 / 锈蚀 / 划痕 mask logic

金属类关键是 metalness 与 rust/paint mask 互斥。锈蚀通常从边缘、凹坑、污渍区域长出；划痕改变 roughness 和 normal，别只改颜色。

base metal → scratch mask → rust mask from curvature/noise → metalness split → roughness up on rust
Meshova`rustyMetal`、`brushedMetal` 已有；应补 curvature-driven wear、paint peel、scratch generator。

SD → SP / Blender / 引擎 export contract

教程常见流程：SD 负责可参数化生成，SP/引擎负责模型局部涂抹和最终展示。Meshova 应明确导出契约：通道名、色彩空间、法线方向、强度范围。

generator params → PBR channels → validation → preview scene → export PNG/JSON → downstream material
Meshova已有 PBR export；应补 material manifest、channel colorspace 标记、normal Y 方向选项。

基础节点体系 node grammar

高频节点不是孤立功能，而是一套语法：shape 产形、transform 变形、blend 合成、histogram 抽 mask、flood fill 做 per-cell 随机、warp 打破机械感。

shape/tile → transform → blend → levels/histogram → flood fill → warp/slope blur → outputs
Meshova可设计“函数式图层管线”：每层有 `height/mask/color/roughnessContribution`，减少随意贴图拼接。

函数 / 参数 / 生成器 procedural API

高级教程反复强调暴露参数：seed、scale、density、wear、warp、roughness、palette。函数节点用于复用随机范围、颜色插值和开关逻辑。

param schema → seeded random ranges → reusable subgraph → expose controls → presets → batch variation
Meshova继续强化 `PRESET_PARAM_SCHEMA`；所有材质 builder 都应有默认参数、范围、语义标签、随机化策略。

生成流程改进建议

面向 Meshova：让 AI 不是“画贴图”，而是选择 preset、调参数、跑评估、迭代。

优先级改进点原因可验证指标
P0材质族 preset schema每类材质的主生成器不同，统一噪声参数不够。材质语义分类通过率；参数随机后仍像同类材质。
P0height/mask-first 管线SD 高质量教程都从 height 和 mask 驱动多通道。normal/ao/baseColor/roughness 结构一致，无通道打架。
P0tiling 质量检测程序化材质最常见失败是接缝和重复 pattern 太明显。边界差分、频谱峰值、平移相关性。
P1per-cell 随机属性砖、瓦、石、鹅卵石、织物花纹都依赖 cell id 随机。同 seed 可复现；cell height/color/roughness 可独立扰动。
P1warp/slopeBlur preset 化大多数自然材质靠变形和侵蚀摆脱机械节点感。生成前后重复度下降；轮廓仍保留目标材质形态。
P1roughness 规则库roughness 不能随便噪声化；湿、锈、布、冰逻辑不同。材质族 roughness 范围符合物理常识。
P1风格化开关写实和风格化材质节点链相似，但频率、色阶、边缘完全不同。同材质可输出 realistic/stylized 两种稳定变体。
P2图片辅助 mask 提取叶片、刺绣、花纹可从图抽结构，但不应直接 bake 成位图终点。输入图只影响语义 mask 和参数，输出仍程序化可编辑。

Meshova 可落地 API 草案

不直接照搬 SD 节点名；用程序化材质函数表达同一逻辑。

`makeLayeredMaterial(layers)`

每层提供 `mask / height / color / roughness / metalness`，最终统一合成 PBR。解决“颜色、法线、粗糙度各做各的”。

`cellSampler(opts)`

`tileSampler` 升级版：返回 cell id、局部 UV、随机属性。砖、石、瓦、刺绣花型共用。

`weavePattern(opts)`

经纬交错、线程宽度、间距、over-under、纤维噪声。布料/麻布/羊毛/服装底纹共用。

`flowStrands(opts)`

方向场驱动发丝、毛发、草、刺绣线方向。输出 height、alpha、direction、color ramp。

`damageMask(opts)`

统一裂缝、剥落、边缘磨损、孔洞。输入 curvature/height/noise，输出多级 mask。

`validatePBR(material)` 扩展

增加语义范围：布料高 roughness、金属 rust 区 metalness 归零、冰雪 normal 不能过硬。
