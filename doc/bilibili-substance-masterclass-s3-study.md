## 总判断

1. P3-P6 应视作一个完整的“科幻硬表面材质生成系统”，不是四个独立小教程。价值最高，覆盖形状语法、语义遮罩、PBR 分层、最终验证闭环。
2. P7 最适合先落地。体量小、视觉收益高，可同时检验 Voronoi、Flood Fill、Bevel、Slope Blur、裂缝、苔藓和调色能力。
3. P2 是 P7 与科幻面板之间的桥梁。重点不是最终“鳞片”外观，而是逐单元处理和稳定裂隙拓扑。
4. P1 对 Meshova 的价值在“方向场驱动材质”，不只是再做一种噪声毛绒。
5. P6 后半段节点新增较少，但很值得学。它证明材质必须在实际 PBR 灯光、球体曲面、近景和多曝光条件下校准，不能只看 2D 贴图。

## 建议复刻清单

### A. 共用基础层，12 项

1. `heightLayerStack`：按优先级组合凸起、凹槽、切口和覆盖层，保留每层语义遮罩。
2. `semanticMaskPack`：统一输出面板、缝隙、边缘、凹腔、螺钉、管线、发光、污损、材质ID。
3. `shapeTransform2D`：形状平移、缩放、旋转、镜像、重复，支持确定性参数。
4. `radialArray`：径向复制扇叶、刻度、螺栓、发光段和通风格栅。
5. `pathStroke`：折线/曲线路径生成带宽度、圆角、端帽、分叉的高度和遮罩。
6. `distanceProfile`：距离场转圆顶、斜坡、台阶、凹槽、倒角和硬边轮廓。
7. `maskedSlopeBlur`：只在指定材质或形状内部做坡度模糊，避免污染邻接区域。
8. `componentFields`：每连通单元生成随机值、渐变、坡度、选择和稳定ID。
9. `frequencyBands`：显式拆分宏观、中观、微观细节，分别控制强度和尺度。
10. `normalLayerStack`：高度法线、机械细节法线、表面微法线按正确方式合成。
11. `maskDebugAtlas`：一次输出所有中间遮罩，支持定位层级和范围错误。
12. `materialLookdevRig`：固定 HDR/程序化环境、曝光、相机、平面/球体/圆柱三种预览体。

### B. P1 定向毛发，9 项

13. 单根毛发轮廓：从窄条、锥形或曲线距离场生成根部到尖端的高度剖面。
14. 输入图驱动散布：位置、密度、旋转、缩放、位移、向量场分别控制毛发实例。
15. 两级毛束：先形成大束流向，再在束内生成细丝，避免均匀白噪声。
16. 方向场弯曲：毛发沿局部向量场连续转向，而不是固定横向拉伸 FBM。
17. 根尖颜色：根部偏暗、尖端偏亮，叠加低频色块和少量离群毛色。
18. 毛束间 AO：束间凹陷产生遮蔽，细丝本身不应把 AO 压黑。
19. 切线各向异性：由方向场生成 `anisotropy` 与 `anisotropyRotation`。
20. 多尺度粗糙度：束级粗糙变化加细丝高光破碎，避免塑料感。
21. 可平铺流场：边界方向和纤维连续，避免 UV 接缝处突然折断。

### C. P2 细胞/鳞片表面，10 项

22. 抖动 Voronoi 细胞：控制密度、长宽比、规则度和种子。
23. 细胞轮廓圆化：距离场、平滑、侵蚀/膨胀组合，得到有机多边形。
24. 单元独立高度：每个细胞获得稳定高度偏移，形成不均匀层次。
25. 单元局部坡度：每片内部具有方向性倾斜或圆顶，而非统一平顶。
26. 单元旋转/缩放随机：随机量以组件ID为源，分辨率变化不跳变。
27. 缝隙宽度场：裂隙有主宽度、局部收窄、分叉和软硬边控制。
28. 边缘破损：沿细胞边界选择性侵蚀，保留完整块与破损块共存。
29. 凹缝沉积：裂隙驱动 AO、污垢、湿度和粗糙度，而非只改颜色。
30. 逐单元调色：按稳定ID做色相、明度、饱和度轻微变化。
31. 表面微损伤：划痕、孔洞、颗粒只影响部分单元，并受尺度约束。

### D. P3-P4 科幻船壳高度与零件语法，19 项

32. 非均匀面板布局：不同尺寸矩形拼接，不使用单一规则网格。
33. 面板接缝：缝宽、深度、倒角和边缘磨损分别输出遮罩。
34. 面板层级：底壳、主面板、覆盖板、凸台至少四级高度优先级。
35. 面板ID随机：每块面板稳定获得高度、颜色和材质变化。
36. 圆形舱口：同心环、中心孔、环形凹槽、外壳倒角组合。
37. 径向涡轮/风扇：扇叶数、角度、宽度、内外半径可参数化。
38. 分段发光环：发光段、间隙、亮度抖动、外圈金属遮罩独立。
39. 圆形通风格栅：横条、径向条和网格三种模式共用壳体。
40. 矩形通风口：凹槽、平行叶片、边框、内部暗腔组合。
41. 管线生成器：直段、弯头、并行管、端点接头和遮挡顺序。
42. 电路走线生成器：细线、直角转折、分叉和端点焊盘。
43. 连接器：壳体、插槽、插针阵列、紧固件和底座。
44. 旋钮/按钮：圆形、D形、方形，多级高度与中心标记。
45. 紧固件库：螺钉、螺栓、铆钉、沉头孔、垫圈，支持径向/矩形阵列。
46. 机械凸台库：圆柱、胶囊、阶梯块、护罩、把手和小型支座。
47. 局部切口：折线切槽、锯齿边、缺口、卡槽，支持布尔式高度组合。
48. 避让/占位遮罩：新零件不能压过舱口、发光件或关键管线。
49. 细节预算：大面板少放细节，视觉焦点周围提升密度，避免全图均匀噪声。
50. 模块子图协议：每个零件返回 `height + normal + masks + bounds + priority`，可组合、可测试。

### E. P5-P6 科幻船壳材质、损伤与验证，13 项

51. 金属基底：金属度稳定，粗糙度保留细微加工变化。
52. 喷漆层：覆盖金属，边缘/划痕处露底，不能让锈蚀区继续保持金属度 1。
53. 面板色彩分组：按面板ID分配主色、辅色和少量强调色。
54. 曲率边缘磨损：边缘亮度、粗糙度、露底宽度分别可控。
55. 凹腔积尘：基于缝隙/AO，再用噪声打散，不是全局 grunge 乘法。
56. 定向划痕：长度、宽度、方向、密度、深度、材质响应分别控制。
57. 锈蚀传播：从露底、接缝、螺钉和水流路径起始，向外扩散。
58. 油污泄漏：从机械接口/通风口附近产生，重力方向拉出拖痕。
59. 雨痕与沉积：方向场控制流动，障碍物下方增强，凸面减弱。
60. 发光遮罩：发光颜色、强度、边缘泛光区和关闭状态分离。
61. 粗糙度叙事：油污更光、积尘更粗、裸金属较光、锈蚀较粗，关系必须一致。
62. 多尺度预览：远看面板节奏，中景看零件层次，近看磨损与微表面。
63. Toolbag/浏览器交叉校准：同一贴图在两个渲染环境检查金属度、法线方向、曝光和色彩空间。

### F. P7 风格化苔藓岩石，11 项

64. Voronoi 岩块骨架：用细胞而非普通 FBM 决定大块面。
65. 岩块高度量化：逐单元高度分级，形成清楚的风格化平面。
66. 单元圆顶/斜面：Flood Fill Gradient/Slope 将平面变成可读岩块。
67. Slope Blur 破形：低频方向噪声打破规则细胞，但保持大轮廓。
68. 裂缝距离场：独立控制裂缝宽度、深度、倒角和内部 AO。
69. 岩块边缘损坏：只破坏部分边缘，保留主形完整性。
70. 岩石分面色：按面朝向、组件ID和高度分配有限色阶。
71. 凹缝苔藓：生长集中于裂缝和潮湿凹腔，不平均覆盖所有表面。
72. 顶面苔藓：结合法线朝上、湿度和噪声，形成第二种生长逻辑。
73. 苔藓厚度与绒感：轻微抬高、提高粗糙度/Sheen，不能只涂绿色。
74. 岩石微颗粒：强度受距离和预览尺度限制，不破坏风格化大面。

## 与 Meshova 现状对照

### 已有，可直接复用

| 能力 | 当前 API |
|---|---|
| Voronoi/平滑细胞 | `voronoi`、`smoothVoronoi` |
| 连通单元场 | `labelComponents`、`floodFillRandom`、`floodFillGradient`、`floodFillHeight`、`floodFillSlope`、`floodFillSelect` |
| 形态与距离操作 | `distanceField`、`bevel`、`dilateMask`、`erodeMask`、`edgeDetect` |
| 破形与滤波 | `warp`、`slopeBlur`、`blur`、`levels`、`histogramScan` |
| PBR 派生 | `heightToNormal`、`normalCombine`、`curvature`、`aoFromHeight` |
| 面板/裂缝/线缆基础 | `panelGenerator`、`crackGenerator`、`cableGenerator` |
| 磨损与天气 | `buildLayeredWearMasks`、`buildWeatherStackMasks`、`applyWeatherStack` |
| 方向纤维基础 | `makeDirectionField`、`makeFiberTensorField`、`plushFur` |
| 苔藓基础 | `mossyStone`、`sceneAwareMossyRock`、`heightBlendMaterials`、`upwardMask` |
| 当前科幻面板 | `sciFiHardSurfacePanel` |

### 当前最大差距

1. `sciFiHardSurfacePanel` 目前主要是规则网格、简单螺钉、通风条和发光线，距离视频中的模块化船壳系统差一层“形状语法”和“零件库”。
2. `plushFur` 仍以拉伸 FBM 为主，缺少毛束层级、向量场散布、根尖变化和各向异性旋转。
3. `sceneAwareMossyRock` 有环境驱动生长，但岩石大形仍偏宏观噪声，缺少 P7 的细胞分面与风格化高度结构。
4. 现有基础函数很多，但缺统一的语义遮罩协议。每个配方独立重算边缘、凹腔、组件ID，会造成重复和不一致。
5. 缺径向结构、路径描边、零件占位避让、层级高度合成四个高复用构件。
6. 缺固定 lookdev 回归。P6 说明材质正确性不仅是数值范围，还包括真实灯光下的读形和材质区分。

## 验收标准

1. 相同 seed、参数、分辨率输出逐字节一致。
2. 512、1024、2048 三档均无平铺接缝，无组件ID跳变。
3. 每个配方输出 Base Color、Normal、Roughness、Metallic、AO、Height；需要时增加 Emission、Anisotropy、Anisotropy Rotation。
4. 中间遮罩可独立导出并命名，禁止只能从最终贴图猜错在哪层。
5. 科幻船壳至少有 12 种可组合零件，支持密度、占位、优先级和语义标签。
6. 远景、中景、近景都能读出不同频段，缩小后不变成均匀噪声。
7. 岩石不依赖颜色也能从 Height/Normal 读出分面；苔藓关闭后岩形仍成立。
8. 毛发关闭 Base Color 后，Normal/Anisotropy 仍能读出连续流向。
9. 锈蚀区域金属度下降；油污粗糙度下降；积尘粗糙度上升；关系符合材质语义。
10. 所有实现仅学习视觉方法与通用算法，自行重写，不复制视频工程文件、节点图或受版权保护资产。

## 快速观看路线

- 只看 2 小时：P7 全部，先理解细胞岩块、裂缝、苔藓闭环。
- 只看 6 小时：P3 前半，重点看面板、圆形舱口、径向风扇和发光环。
- 只看材质：P5 中后段，关注面板ID、露底、锈、油、尘、粗糙度。
- 只看工程化：P4 的零件库组织 + P6 的输出和 Toolbag 校准。
- 完整学习：P3 -> P4 -> P5 -> P6；P2/P7 补细胞系统；P1 最后补方向材质。

本地联系表按分P编号命名，每张 5x5。P3-P6 每 8 分钟采样；P1、P2、P7 至少保留 24 个均匀采样点，可先用联系表定位，再打开原视频精看。
